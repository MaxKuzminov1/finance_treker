# 💰 FinancePro — Система финансового учета

**FinancePro** — это десктопное приложение для комплексного учета доходов и расходов малого бизнеса. Программа предоставляет инструменты для ведения транзакций, планирования бюджетов, управления справочниками и глубокой аналитики с генерацией PDF-отчетов.

## 🛠 Технологический стек

* **Язык:** Python 3.11+
* **GUI Фреймворк:** PyQt6
* **База данных:** MySQL (взаимодействие через `mysqlclient`)
* **Аналитика и графики:** Matplotlib, Pandas, SciPy
* **Отчетность:** ReportLab (генерация PDF), OpenPyXL (Excel)
* **Сборка:** PyInstaller

---

## 📦 Основные модули

Приложение построено по модульной архитектуре. Каждый модуль независим и отвечает за свою бизнес-логику:

1. **📊 Учет операций (`Module1Widget`)**
   * Ввод доходов и расходов с привязкой к категориям и контрагентам.
   * Учет статусов оплат (Оплачено, Частично, Создана).
   * Продвинутая фильтрация и поиск.
   * Импорт и экспорт данных в форматах CSV и Excel.

2. **💰 Управление бюджетом (`Module2Widget`)**
   * Установка лимитов и планирование бюджетов на месяц/квартал/год.
   * План-фактный анализ с графическим отображением отклонений.
   * Умное распределение крупных сумм.

3. **📈 Аналитика (`Module3Widget`)**
   * Интерактивные дашборды с ключевыми метриками (KPI).
   * Графики Cash Flow и структуры расходов (Matplotlib).
   * Налоговый радар и P&L симулятор (What-If анализ).
   * Экспорт аналитики в детализированные PDF-отчеты и Excel.
   * *Интеграция:* Поддержка Drill-down (переход по клику из графика в модуль операций с сохранением фильтров и дат).

4. **📚 Справочники (`Module4Widget`)**
   * Управление иерархическим деревом категорий (с поддержкой макро-группировок и слияния).
   * База контрагентов.
   * Глобальные настройки приложения (выбор темы, формата дат).
